## Overview

This repository contains everything needed to deploy and operate my homelab. It serves as the single source of truth for the infrastructure, services, and automation running on my home network.

## What's in here

- **Infrastructure provisioning** — declarative configuration for machines, VMs, and containers
- **Service deployments** — definitions for self-hosted applications and services
- **Networking** — DNS, reverse proxy, firewall rules, and VPN configuration
- **Automation** — scripts and pipelines for routine operations and maintenance
- **Monitoring & alerting** — observability stack setup

## Goals

- Keep the homelab reproducible: any machine should be rebuildable from this repo
- Document the "why" behind infrastructure decisions, not just the "what"
- Minimize manual steps and snowflake configurations
